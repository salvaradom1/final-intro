API Docs: GameFinder

Esta API REST permite gestionar información sobre juegos, consolas y DLCs relacionados. Cada entidad tiene sus propias operaciones CRUD, y las entidades están interrelacionadas, lo que permite consultar y actualizar relaciones entre juegos y consolas, así como juegos y DLCs.

/GET

Solicita a la base de datos la información almacenada de cada endpoint.

/POST

Permite cargar a la base de datos un juego, consola o DLC.

/PUT

Permite editar información existente.

/DELETE

Elimina la información solictada.
